<h1>Razer.com Clone</h1>

Deployed Link -: https://razer-clone-gamma.vercel.app/

Tech Stack used:-
1. React
2. Redux
3. Middleware
4. Chakra UI
5. CSS
6. json server
7. Routing
8. Google Recaptcha
9. Redux-Thunk



Pages are -:
1. Home
2. Product
3. Product Details
4. Sign Up
5. Sign in
6. Payment
7. Congratulations After Sucessfull order
